Overview

LuaSOAP is a library to ease the use of SOAP. LuaSOAP provides a very simple API that convert Lua tables to and from XML documents.

LuaSOAP is free software and uses the same license as Lua 5.0.

Current version

Current version is 1.0 alpha. It was developed for Lua 5.0 based on LuaExpat.

Installation

LuaSOAP is a Lua library composed by two files: soap.lua and http.lua. Both must be copied to a directory named soap created in your LUA_PATH. That way one could use require "soap/soap" to load the module. Other simple way to make it work is to add a rule like .../?/?.lua... in your LUA_PATH. This way one could use just require "soap".
